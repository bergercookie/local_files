DAMA^{ROB} Toolbox: Installation

    UNIX and Windows Systems: just launch dama_install.m script.
    On UNIX Systems you need root privileges to install the toolbox.
    Please remark that installation script can ask you root password up to three times.
    This is a typical successful output of dama_install.m script on UNIX systems:

>> dama_install

running install_lin.sh script 
 this will need root privileges. 
 Note that root password will not be stored. 
 Please insert root password or
 press just enter to break installation

    If you have a MAC or if you encounter some trouble with automatic installation, follow manual installation steps.


MANUAL INSTALLATION:

    Copy 'damarob' folder into Matlab™ Toolbox folder:
    – on Unix systems it is something like: /usr/local/matlabR2007a/toolbox (depending on Matlab™ versions)
    – on Windows systems it should be: C:\Program Files\MATLAB\R2007a\toolbox (depending on Matlab versions)

    note that folder name may change.
    On Unix systems, assure that you have permissions to write into destination folder, and permissions to write the pathdef.m file located in: TOOLBOXDIR/local. (this folder could be something like: /usr/local/matlabR2007a/toolbox/local)

    Add damarob path and subfolders to Matlab™ path:
    – menu File → Set Path → Add with Subfolders, then select the folder you copied at previous step.
    – alternatively you can use the following commands:

[ON UNIX SYSTEMS]
addpath /usr/local/matlabR2007a/toolbox/damarob/
addpath /usr/local/matlabR2007a/toolbox/damarob/doc/
addpath /usr/local/matlabR2007a/toolbox/damarob/examples/
addpath /usr/local/matlabR2007a/toolbox/damarob/mfiles/
addpath /usr/local/matlabR2007a/toolbox/damarob/simulink/
savepath

[ON WINDOWS SYSTEMS]
addpath 'C:\Program Files\MATLAB\R2007a\toolbox\damarob\'
addpath 'C:\Program Files\MATLAB\R2007a\toolbox\damarob\doc\'
addpath 'C:\Program Files\MATLAB\R2007a\toolbox\damarob\examples\'
addpath 'C:\Program Files\MATLAB\R2007a\toolbox\damarob\mfiles\'
addpath 'C:\Program Files\MATLAB\R2007a\toolbox\damarob\simulink\'
savepath

Note that the folder name may change (depending on Matlab™ versions).
