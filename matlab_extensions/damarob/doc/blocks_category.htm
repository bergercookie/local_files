Kinematics Blockset:


Transpose Jacobian Inverse Kinematics

The block computes the inverse kinematics of a trajectory in the operational space for a given manipulator.


Direct Kinematics

The block computes the direct kinematics of a manipulator. Output signals are nsap vectors, that is the first three columns of the Tbe homogeneous matrix (last row excluded).


Change Frame

The block computes the coordinates of a vector in different frames.


Manipulator Dynamics Blockset:


Manipulator Dynamics

The block evaluates accelerations, velocities and positions of a manipulator's joints from generalized forces applied to joints (tau) and the generalized forces applied to the end effector (he_ext).
Don't forget to connect the second input with an “inertia load” block, even if you don't want to simulate added inertia (in that case set all inertia to null values).


Inertial Load

The block creates the inertia matrix of a mass load, given its mass and its inertia tensor.
